# 診断士 Practice

中小企業診断士1次試験のジャンル別問題と過去10年分の公式過去問PDF演習をまとめたPWAです。

## 機能

- 1次試験7科目の各科目200問、合計1,400問の4択クイズ
- 公式PDFから抽出した過去問を、クイズとは別の「過去問演習」タブに分離
- 正誤判定、解説、弱点復習、フラグ付き問題
- 令和7年度から平成28年度までの1次試験7科目を一覧化
- 問題PDFと正解・配点ページへの直接リンク
- 過去問PDFの演習済み、復習対象をブラウザ内に保存
- USCPA Vocab Quiz に近いタブ型UI
- 学習データのJSONエクスポート/インポート
- PWA manifest と service worker を同梱

## 注意

過去問抽出問題は、図表依存を除いた一部の問題文・選択肢・正解を公式PDFから抽出して収録しています。学習用クイズとは別データとして扱っています。PDFおよび正解表は、日本中小企業診断士協会連合会が公開する資料へのリンクです。
