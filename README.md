# 診断士 過去問演習

中小企業診断士1次試験の過去10年分を、年度・科目別に開いて学習記録を残せるPWAです。

## 機能

- 令和7年度から平成28年度までの1次試験7科目を一覧化
- 問題PDFと正解・配点ページへの直接リンク
- 演習済み、復習対象、得点、メモをブラウザ内に保存
- 年度、科目、状態、メモ検索で絞り込み
- 学習データのJSONエクスポート/インポート
- PWA manifest と service worker を同梱

## 注意

過去問本文はアプリ内に複製していません。PDFおよび正解・配点は、LEC東京リーガルマインドの過去問ダウンロードサービスおよび日本中小企業診断士協会連合会が公開する資料へのリンクです。
